## Requirements

Swift 4, Xcode 9, iOS 10.0+

## Installation

```bash
git clone https://github.com/pprivalov/Swifty-Companion.git ~/SwiftyCompanion
cd ~/SwiftyCompanion
open SwiftyCompanion.xcworkspace
```
